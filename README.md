# 乳鴿達人網站設計 — 任務記錄
_時間：2026-04-18 17:32 GMT+8_

## 目標
設計一頁式零售網站「乳鴿達人」，主題為宮廷藥膳乳鴿，配色精緻，含藥膳影音區塊。

## 美學方向
- **風格**：高端中式奢華 × 現代極簡
- **配色**：深酒紅 (#7B1C2A / #4E0F18) + 金色 (#C9A84C) + 象牙白 (#FAF6EE)
- **字體**：Noto Serif TC（標題）+ Cormorant Garamond（裝飾英文）+ Noto Sans TC（內文）
- **特效**：自訂游標、噪點紋理、滾動揭示動畫、視差裝飾圓環

## 頁面結構
1. **固定導覽列** — Logo + 錨點連結 + CTA 按鈕
2. **Hero 區** — 雙欄佈局，左側深酒紅文字區，右側裝飾性大字
3. **品牌故事** — 三代傳承敘事 + 四項數據統計
4. **招牌菜單** — 6款藥膳產品卡片（深色背景網格）
5. **藥膳影音** — 1大4小影片縮圖佈局（含播放按鈕）
6. **藥材精選** — 4種核心藥材介紹 + 書法大字裝飾
7. **製作工藝** — 四步驟流程圖
8. **顧客好評** — 3則評價卡片
9. **CTA 橫幅** — 金色背景，首購優惠
10. **頁尾** — 品牌介紹 + 四欄連結 + 社群媒體
